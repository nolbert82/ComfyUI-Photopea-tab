# ComfyUI Photopea Tab

A custom extension for ComfyUI that integrates **Photopea**, a powerful web-based photo editor, directly into the sidebar.

Current version: 1.1.0

Four new buttons have been added when clicking on images nodes :
- Open in Photopea
- Import from Photopea
- Import selected layer
- Import selected layer as mask

You can also:
- Hide ads via a toggle
- Zoom in-and-out
- Maximize the page's width
- Toggle Fullscreen
